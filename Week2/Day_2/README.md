# Number Manipulation Functions in MySQL

## Overview
This module covers MySQL numeric functions used for mathematical operations, rounding, formatting, and numeric analysis.

The exercises focus on salary calculations, financial operations, and numeric transformations.

---

## Topics Covered

- ABS()
- ROUND()
- CEIL()
- FLOOR()
- TRUNCATE()
- MOD()
- POWER()
- POW()
- SQRT()
- SIGN()
- RAND()
- FORMAT()
- GREATEST()
- LEAST()
