# String Functions in MySQL

## Overview
This module covers MySQL string functions used for text manipulation, formatting, extraction, and transformation.

The exercises include employee and customer data processing using various string-based operations.

---

## Topics Covered

- LENGTH()
- CHAR_LENGTH()
- UPPER()
- LOWER()
- TRIM()
- LTRIM()
- RTRIM()
- CONCAT()
- CONCAT_WS()
- SUBSTRING()
- SUBSTR()
- LEFT()
- RIGHT()
- INSTR()
- LOCATE()
- REPLACE()
- REVERSE()
- LPAD()
- RPAD()
- FIND_IN_SET()
