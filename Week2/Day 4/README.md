# REGEX Foundation Assessment

## Overview
This assessment focuses on applying REGEX concepts in practical SQL scenarios using pattern matching and text extraction techniques.

The exercises involve extracting numbers, alphabets, country codes, email components, and structured values from mixed data.

---

## Topics Covered

- Numeric extraction
- Alphabet extraction
- Country code extraction
- Email parsing
- Pattern matching
- Exact character matching
- Continuous sequence extraction
- REGEXP_SUBSTR()

---

## Practice Areas

- Phone number parsing
- Email domain extraction
- Mixed string analysis
- Structured identifier parsing
- Data cleaning and validation
- Regex-based transformations

## Learning Objectives

- Apply regex in real-world SQL problems
- Extract structured information from text
- Improve pattern recognition skills
- Strengthen regex query writing
- Perform text-based data validation
