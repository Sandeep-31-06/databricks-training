# REGEX Foundation in MySQL

## Overview
This module introduces Regular Expressions (REGEX) in MySQL for pattern matching, validation, and text extraction.

The exercises focus on extracting digits, alphabets, email patterns, and validating structured text data.

---

## Topics Covered

- REGEXP
- REGEXP_SUBSTR()
- Character classes
- Digit extraction
- Alphabet extraction
- Start and end anchors
- Quantifiers
- Pattern matching
- Email validation
- String extraction

---

## Practice Areas

- Email validation
- Digit extraction
- Domain extraction
- Pattern recognition
- Data quality checks
- Structured text validation
