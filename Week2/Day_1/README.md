# NULL Functions in MySQL

## Overview
This module focuses on handling NULL values in SQL using different MySQL NULL functions and conditions.

The exercises include NULL replacement, NULL-safe calculations, filtering missing data, and real-world NULL handling scenarios.

---

## Topics Covered

- IFNULL()
- COALESCE()
- NULLIF()
- IS NULL
- IS NOT NULL
- NULL-safe calculations
- Default value replacement

---

## Practice Areas

- Replacing NULL values
- Employee salary handling
- Product stock validation
- Order payment calculations
- NULL filtering
- Divide-by-zero prevention
