# Date Manipulation Functions in MySQL

## Overview
This module focuses on MySQL date and timestamp functions used for handling, formatting, filtering, and analyzing date-based data.

The exercises include real-world business scenarios involving orders, delivery tracking, financial year calculations, and timestamp analysis.

---

## Topics Covered

- CURDATE()
- NOW()
- CURRENT_TIMESTAMP
- YEAR()
- MONTH()
- DAY()
- EXTRACT()
- MONTHNAME()
- DAYNAME()
- DATE_ADD()
- DATE_SUB()
- DATEDIFF()
- TIMESTAMPDIFF()
- DATE_FORMAT()
- STR_TO_DATE()
- LAST_DAY()
- Financial Year Logic
- Weekday and Weekend Identification

---

## Practice Areas

- Order delivery analysis
- Date filtering
- Weekend detection
- Timestamp formatting
- Financial year calculations
- Date arithmetic
- Real-time date scenarios

- Perform date-based calculations
